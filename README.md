## web-template 前端模板项目

#### 项目简介

lego 组件库, 有vue, react 两个分支版本，目的支持项目快速组件化开发。

产出为 umd 格式，支持全局或者模块化引入。

应用场景除了后台管理系统，还支持解压组件源码(/output/lego.zip)至项目内定制化开发，减少从头开发组件的开销并提供一个初始不错的质量，此场景适用于前台型系统和后台管理类系统。
